# Task Manager App

A full stack task manager built with the MERN stack.

## Features
- Add tasks
- Mark tasks complete/incomplete
- Delete tasks
- Data stored in MongoDB

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js

## How to Run
1. Clone the repo
2. cd backend → npm install → node server.js
3. cd frontend → npm install → npm run dev
